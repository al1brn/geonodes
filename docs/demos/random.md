# random demo

![render](images/random_render_01.png)

All the random generators provide a _Single_ output socket which can be used on a single value
input socket.

## Modifiers

- **Random Uniform Value** : equivalent to standard node plus a _Single_ output socket
- **Random Uniform Vector** : equivalent to standard node plus a _Single_ output socket
- **Random Normal Value** : random value on a normal distribution
- **Random Normal Vector** : random direction with a random normal length 
- **Random Rotation** : random axis with a random normal rotation angle
