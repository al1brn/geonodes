## Initialization

```python
from geonodes import nodes
node = nodes.RgbCurves(fac=None, color=None, label=None, node_color=None)
```



## Arguments


### Input sockets

- fac : Float
- color : Color

### Node label

- label : Geometry node display label (default=None)
- node_color : Geometry node color (default=None)

## Output sockets

- color : Color
